CertiLens

Chrome Extension · Manifest V3 · Open Source (GPL-3)

6Engines
~2sScan Time
0API Keys
MV3Compliant

CertiLens is a real-time phishing detection Chrome extension that runs six independent security engines in parallel — Certificate Transparency (crt.sh), domain age via RDAP, DNS email security (SPF/DMARC over DoH), HTTP security header audit, DOM heuristics, and homograph detection. Unlike blocklist-based tools, it actively analyzes sites it has never seen before, catching brand-new phishing domains before any database knows they exist. Each engine contributes an additive penalty to a 0–100 risk score, and every finding is surfaced with a clear rationale — no black boxes.

Credora

CompTIA Study Guide · Web App

Credora is a web-based CompTIA study guide covering A+ (220-1101/1102), Network+ (N10-009), and Security+ (SY0-701). It organizes exam content into searchable, structured notes with domain breakdowns — designed to be exam-ready and easy to navigate. The platform covers everything from hardware fundamentals to cryptography, identity management, and incident response across all three certifications.
